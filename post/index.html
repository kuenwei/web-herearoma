如何付款給“循香薈”

在此聲明，此篇並非有任何委託，代表本文非業配文，單純因為我使用了這些功能。身為一個獨立經營者，一直希望能夠用好的方法提供給大家便利的付款手續（信用卡以及其他多元付款方式），給買方與賣方更多保障，所以寫了這一篇付款工具的索引文章。希望更多人一同享有方便、安全的的付款經驗。

這次更新這一篇，主要是我在付款的方式做了一個很大的改變，原則只有一個，我希望找到簡單卻又能夠盡量沒有手續費的收款方式。一方面又要顧及普及性，且現實環境我可用的資源等，最後從傳統到新潮，從繁瑣到簡便的方式都融合如下三個大項目，先以表格呈現優缺點與推薦度，下方的文章則分別介紹使用方式。

第三方支付線上金流平台

第三方支付一直都是我很推薦的方式，之前其實很想推廣銀行推出的第三方支付，我覺得這是目前看到最值得推薦的付款方式之一，一方面是安全（交易緩撥、款項保管），同時也能夠支援信用卡付款，雖然需要跟收款方（也就是我）收取手續費，但經過評估，還是覺得免對帳且給大家的安心付款環境很值得，推薦大家都使用這個方式付款。

多元的付款方式

  • 信用卡：透過各家信用卡付款，會酌收手續費。
  • ATM 轉帳：實體 ATM、網路 ATM、網路銀行轉帳，如果非合作銀行需要自己負擔轉出手續費。（想知道合作銀行是哪些？）
  • 超商代碼/條碼：付款金額不逾 2 萬，可至全台（7-11、OK超商、萊爾富）店內之多媒體機台（ibon、OK-go、Life-ET）上列印繳費單至超商櫃台以現金繳費；或透過印表機列印條碼付款單，印單後請至四大超商櫃檯付現。。

QR Code 掃碼轉帳

先在這邊提醒大家，QR Code 均是要使用手機特定的 APP 掃碼才能使用喔！想使用手機掃碼付款的朋友，請協助先下載相對應的 APP，並確認已經開立完成轉帳帳戶了。
