EO 能量整復

Energetic Osteopathy，由 Kalu Schreiber 創始，中文名稱為「能量整復」。在能量中，我們重新感受身體的韻律，並由施作者一同陪伴身體、器官的韻律舞動，在協作中一同感受身心平衡的氛圍。

SRT 靈性反應療法

靈性反應療法(Spiritual Response Therapy, SRT)由 Robert Detzler 創辦，是一個通過搜尋潛意識與靈魂紀錄，快速找出和釋放當中不和諧、限制性的思想並代之以慈愛、支持的思想和信念的系統。SRT提供正確、力量強大、幾乎無痛並精準的方式去改變我們的內心和外在生命。從而讓我們生活得更自在。SRT系統使用一個靈擺，一副圖表及一些特定的問題來搜尋和清理潛意識與靈魂紀錄中的所有不和諧程式。
